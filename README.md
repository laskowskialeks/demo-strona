# Salon Nova Demo Website

Salon Nova Demo Website is a modern landing page for a fictional hair salon.

The project was created as an HTML, CSS and JavaScript practice website focused on layout, responsive design and presenting a service-based business online.

Live demo: https://laskowskialeks.github.io/demo-strona/

## About the project

The website presents a fictional hair salon called Salon Nova.

The page includes information about services, prices, customer reviews, contact details and online booking. The design is clean, elegant and focused on helping visitors quickly understand the offer and contact the salon.

The website is built as a simple business landing page and can be used as a demo project for a local service company.

## How to run locally

1. Clone the repository:

```bash
git clone https://github.com/laskowskialeks/demo-strona.git
```

2. Open the project folder:

```bash
cd demo-strona
```

3. Open the `index.html` file in your browser.
